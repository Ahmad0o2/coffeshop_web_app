# Client App

Frontend for the Cortina.D Coffee House web app.

## Stack

- React 19
- Vite
- Tailwind CSS
- React Router
- React Query
- Axios
- Socket.IO Client
- Framer Motion

## What The Client Covers

- Home page and homepage media sections
- Menu and product detail pages
- Cart and checkout flow
- Order history and live order status
- Rewards and redemption flow
- Events, gallery, and location pages
- Admin dashboard UI

## Environment

Create:

```text
client/.env
```

From:

```text
client/.env.example
```

Required variables:

```env
VITE_API_URL=http://localhost:5000/api/v1
VITE_SOCKET_URL=http://localhost:5000
VITE_GALLERY_IMAGES=
```

## Install

```bash
cd client
npm install
```

## Run In Development

```bash
npm run dev
```

Default local URL:

```text
http://localhost:5173
```

## Available Scripts

```bash
npm run dev
npm run build
npm run lint
npm run preview
```

## Important Notes

- API requests are configured in `client/src/services/api.js`
- Auth token is read from `localStorage`
- Socket server URL comes from `VITE_SOCKET_URL`
- The client expects the backend API under `/api/v1`

## Main App Entry Points

- `client/src/main.jsx`
- `client/src/App.jsx`
- `client/src/layouts/MainLayout.jsx`

## Key Folders

- `client/src/pages` Route-level screens
- `client/src/components` Shared UI and feature components
- `client/src/context` Auth, cart, and theme state
- `client/src/hooks` Shared hooks
- `client/src/services` API setup
- `client/src/utils` Pricing, inventory, and API helpers

## Build Notes
